Randomator - program which take the string from input file ("input.txt") and interchanges two sysmbols (random positions) 
in it until the string will be as the source string. The number of total symbols interchanges saves in output file ("output.txt").

Randomator - программа которая берет строку из исходного файла ("input.txt") и переставляет два рандомных символа в ней местами
до тех пор пока строка не вернется к исходному виду. Общее количество перестановок символов сохраняется в файл вывода ("output.txt").
